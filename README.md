# Event+ Front-end: Interface de Gestão de Eventos

A **Event+ Interface** é o front-end de um sistema de gerenciamento de eventos institucionais, desenvolvida com foco em usabilidade e performance. O projeto faz parte do portfólio do **Curso Técnico em Desenvolvimento de Sistemas**, demonstrando habilidades avançadas em desenvolvimento **Front-end** com React.

## 💻 Tecnologias e Arquitetura

O desenvolvimento seguiu padrões modernos da indústria para garantir a manutenção e performance do sistema:

| Categoria | Tecnologia |
| :--- | :--- |
| **Framework** | React |
| **Linguagem** | JavaScript / JSX |
| **Estilização** | CSS Modules / Styled Components |
| **Roteamento** | React Router Dom |
| **Consumo de API** | Axios |

## 🛠️ Funcionalidades Principais

A interface permite ao usuário interagir com o sistema de forma intuitiva:

*   **Página de Login:** Autenticação segura com JWT.
*   **Gestão de Eventos:** Visualização, criação, edição e exclusão de eventos.
*   **Interatividade:** Sistema de comentários e confirmação de presença.
*   **Responsividade:** Interface adaptada para diferentes tamanhos de tela.

## 🎓 Contexto Acadêmico

Este projeto consolidou conhecimentos em **React**, **Consumo de APIs**, **Gerenciamento de Estado** e **Roteamento**, pilares fundamentais para o desenvolvimento de aplicações front-end modernas e escaláveis.
